Steps for Execution:
1) Put the test cases in "instances" folder with this convention "<filename>.graph".
2) Execute ./run_script.sh
3) run script includes the following:
	a) compile command
	b) outputs the executable as Minfill_Ordering
	c) executes testcase files (named *.graph inside folder instances ) one by one
4) Keep "instances" folder, Minfill_Ordering and run_script.sh in same folder
